Raphael JS Demo

Raphaël is a small JavaScript library that should simplify your work with vector graphics on the web. If you want to create your own specific chart or image crop and rotate widget, for example, you can achieve it simply and easily with this library.

Common Needs for Web Graphics:

  • DOM element transforms: CSS3
  • Page layouts and scaffolding: CSS3
  • Imported SVGs: SVG, CSS3 (cannot be manipulated by Raphael, but there are extensions to do this)
  • On-the-fly custom shapes (charts) and animations: Raphael

Benefits of Raphael:

  • Backwards compatibility: Degrades to VML vector graphics in older browsers (IE 8)
  • Based on SVG: infinite scaling, and greater flexibility with declaring draw paths and fills (lines, arcs, bezier arcs, etc)
  • Easily create SVG shapes and animations without having to import

Why NOT use SVG?

  • Rendering thousands of nodes, more performant to do rendering in canvas (no vector math… paints mapped pixels)
  • Requires support for IE8 - vector fallback is VML
  • Cannot manipulate existing HTML/non SVG shapes
